欢迎阅读本项目的Markdown写作指南！本文将详细介绍如何在本项目中编写规范的Markdown文件。

一、Front-matter配置

每个Markdown文件开头都需要包含Front-matter配置，使用YAML格式，包裹在---之间。

四、注意事项

  1. 文件名使用英文或数字，避免使用中文和特殊字符
  2. 日期格式使用YYYY-MM-DD
  3. 标签和分类使用小写，多个单词用连字符连接
  4. 图片建议使用相对较小的尺寸，以提高加载速度
  5. 代码块要指定正确的语言，以便语法高亮
